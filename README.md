## 🚀 **GET STARTED IN 3 COMMANDS**

```bash
# 1️⃣ Clone this masterpiece
git clone https://github.com/QuaDDom/portfolio.git

# 2️⃣ Enter the realm
cd portfolio

# 3️⃣ Install the magic
npm install
```

## 🎮 **USAGE**

```bash
npm run dev
```

### Production Build (Ready for Prime Time)

```bash
npm run build
npm start
```

```bash
npm run format
```
